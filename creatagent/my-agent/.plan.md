# Plan: 创建前端聊天页面

## 方案

使用 JDK 内置 `com.sun.net.httpserver.HttpServer`（无需额外依赖），创建一个简单的 Web 服务器 + 前端聊天页面。

## 架构

```
浏览器 (chat.html)
    ↓ POST /api/chat  { message: "..." }
Java HttpServer (ChatServer.java)
    ↓ 调用 agents-flex 同步 chatLoop
MiMo API
    ↓ 返回结果
HttpServer → JSON 响应 → 浏览器展示
```

## 需要创建的文件

### 1. `src/main/java/com/myagent/ChatServer.java`
- JDK HttpServer 监听 `localhost:8080`
- `GET /` → 返回 chat.html 页面
- `POST /api/chat` → 接收消息，调用 agent chatLoop，返回 JSON 响应
- `GET /api/history` → 返回对话历史
- `POST /api/clear` → 清空对话历史
- 静态文件从 `src/main/resources/static/` 提供

### 2. `src/main/resources/static/chat.html`
- 单文件 HTML（内嵌 CSS + JS）
- 聊天气泡界面（用户消息右侧蓝色，AI 消息左侧灰色）
- 支持 Markdown 渲染（AI 回复含代码块、表格等）
- 显示加载状态（"AI 思考中..."）
- 支持回车发送
- 清空对话按钮

## 不需要修改的文件
- `App.java` — 保留 CLI 版本不变
- `pom.xml` — 无需新增依赖
- `skills/` — 不变

## 运行方式
```powershell
mvn -q compile exec:java -Dexec.mainClass="com.myagent.ChatServer"
# 浏览器打开 http://localhost:8080
```
