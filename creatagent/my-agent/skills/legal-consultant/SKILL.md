# 法律咨询技能

## 功能描述
提供专业的法律咨询服务，包括但不限于：
1. 法律问题解答：针对用户提出的法律问题提供专业分析和建议
2. 合同审查：审查合同条款，识别潜在法律风险
3. 法规解读：解读相关法律法规，说明其适用性和影响
4. 案例分析：分析相关案例，提供参考意见
5. 法律风险评估：评估特定行为或决策的法律风险

## 触发条件
当用户提出以下类型的问题时触发：
- 法律咨询问题
- 合同审查请求
- 法规解读需求
- 案例分析请求
- 法律风险评估

## 输入输出
### 输入
- 用户的法律问题或咨询内容
- 需要审查的合同文本或关键条款
- 需要解读的法律法规或条款
- 需要分析的案例信息

### 输出
- 专业法律分析：基于现行法律法规的分析
- 风险提示：指出潜在的法律风险
- 建议方案：提供可行的解决方案或建议
- 法律依据：引用相关法律法规或案例
- 注意事项：提醒用户需要注意的关键点

## 使用说明
1. 明确问题：清晰描述您的法律问题或需求
2. 提供背景：提供必要的相关背景信息
3. 上传文档：如需合同审查，请上传合同文本或关键条款
4. 指定领域：说明问题涉及的法律领域（如合同法、劳动法、知识产权等）

## 注意事项
- 本技能提供的信息仅供参考，不构成正式法律意见
- 对于复杂或重大的法律问题，建议咨询专业律师
- 法律法规可能随时间变化，请以最新官方信息为准
- 不同地区的法律规定可能存在差异
