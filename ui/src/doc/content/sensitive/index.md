# 敏感词管理

敏感词管理模块是企业级智能体应用的核心安全组件，通过系统化的关键词规则配置与实时内容扫描，在智能体交互全流程中构建多层次的内容安全防线。

## 功能概述

- 支持按分类管理敏感词配置，同一分类下配置名称唯一
- 与智能体关联，支持在智能体层面启用/禁用敏感词过滤
- 删除/修改配置时自动触发关联智能体重新注册

## 核心概念

### 分类

分类是敏感词配置的组织维度，用于将不同领域或用途的敏感词进行归类管理。例如：`政治敏感`、`色情暴力`、`商业机密` 等。

- 同一分类下的配置名称必须唯一
- 分类按需自定义，创建配置时可直接新增分类
- 页面顶部支持按分类快速筛选

### 处理动作

| 动作 | 说明 | 额外配置 |
|------|------|----------|
| **警告（WARN）** | 检测到敏感词后发出警告提示，但不阻断对话 | 无 |

### 敏感词列表

每个敏感词配置包含一组敏感词关键词，支持以下输入方式：

- 输入框输入后按回车添加
- 支持批量输入：使用空格、逗号或顿号分隔多个词
- 自动去重，避免重复添加
- 支持单个删除和一键清空

## 操作指南

### 创建敏感词配置

1. 在敏感词管理页面点击 **添加新敏感词配置** 卡片
2. 填写表单：
   - **分类**：选择已有分类或输入新分类名称
   - **名称**：配置名称（同一分类下唯一）
   - **描述**：配置说明
   - **敏感词列表**：在输入框中输入敏感词，按回车添加
   - **替换文本**：当处理动作为"替换"时，设置替换内容
3. 点击确认提交

### 查看详情

点击卡片右上角菜单 → **查看**，可查看配置的完整信息，包括：

- 关联的智能体列表
- 分类、名称、描述
- 敏感词列表

### 编辑配置

点击卡片右上角菜单 → **编辑**，修改配置内容。

:::warning
修改敏感词配置后，所有关联该配置的智能体将自动重新注册，以确保新规则立即生效。
:::

### 启用/禁用

点击卡片右上角菜单 → **启用/禁用**，切换配置的启用状态。

:::warning
禁用被智能体引用的敏感词配置时，系统会进行二次确认，提示影响的智能体列表。
:::

### 删除配置

点击卡片右上角菜单 → **删除**，删除该配置。

:::warning
删除被智能体引用的敏感词配置时，系统会进行二次确认。确认删除后，关联智能体的敏感词过滤将失效。
:::

### 搜索与筛选

- **分类筛选**：页面顶部分段控制器切换分类
- **名称搜索**：输入关键词后回车搜索

## 在智能体中使用

在智能体配置的「工具与能力」步骤中，可以关联敏感词配置：

1. 选择一个敏感词配置（每个智能体只能关联一个）
2. 开启 **启用敏感词过滤** 开关
3. 开启后，智能体将对输入和输出内容进行敏感词过滤

## 最佳实践

### 分类设计建议

- 按风险等级分类：如 `高危`、`中危`、`低危`
- 按业务领域分类：如 `金融合规`、`医疗隐私`、`个人信息`
- 避免分类过多，建议 3~8 个分类

### 处理动作选择

- **警告**：适用于边界内容，提醒用户注意措辞

### 敏感词维护

- 定期审查和更新敏感词列表
- 避免过度添加导致误伤正常对话
- 同义词、变体词应一并添加
