# 提示词模板管理

系统提示词是智能体的"角色定义与行为准则"，通过结构化指令为大模型设定明确的角色定位、能力边界和交互范式。精心设计的提示词模板能够显著提升智能体在特定领域的表现一致性、专业性及安全性。


## 功能概述

- 支持按分类管理提示词模板
- 提供模板内容编辑器（Markdown 语法）
- 智能体可选择模板并支持"随模板变化"自动更新
- 修改/删除模板时自动触发关联智能体重新注册


## 核心概念

### 模板与自定义提示词

智能体的系统提示词有两种来源：

| 模式 | 说明 | 更新方式 |
|------|------|----------|
| **模板模式** | 选择已有模板，提示词内容来自模板 | 随模板更新自动同步 |
| **自定义模式** | 不使用模板，手动编写提示词内容 | 手动修改 |

### 随模板变化

当智能体关联了提示词模板后，可以开启 **随模板变化** 开关：

- **开启**：提示词内容始终与模板保持一致，模板更新后智能体自动使用最新内容
- **关闭**：选择模板时会将模板内容复制到系统提示词中，之后可自由编辑，不再随模板变化


## 操作指南

### 创建提示词模板

1. 在提示词模板管理页面点击 **添加新提示词模板** 卡片
2. 填写表单：
   - **分类**：选择已有分类或输入新分类名称
   - **名称**：模板名称
   - **描述**：模板用途说明
   - **内容**：使用 Markdown 编辑器编写提示词内容
3. 点击确认提交

### 查看详情

点击卡片右上角菜单 → **查看**，可查看模板完整信息，包括关联的智能体列表。

### 编辑模板

点击卡片右上角菜单 → **编辑**，修改模板内容。

:::warning
修改模板后，所有开启"随模板变化"的关联智能体将自动使用新内容并重新注册。
:::

### 删除模板

点击卡片右上角菜单 → **删除**。

:::warning
删除被智能体引用的模板时，系统会进行二次确认。
:::

### 搜索与筛选

- **分类筛选**：页面顶部分段控制器切换分类
- **名称搜索**：输入关键词后回车搜索


## 在智能体中使用

在智能体配置的「模型与提示词」步骤中：

1. 选择分类，浏览该分类下的模板列表
2. 选择一个提示词模板（Radio 单选）
3. 配置 **随模板变化** 开关
4. 若关闭随模板变化，可在下方编辑器中自定义提示词内容

## 提示词编写技巧

### 结构化模板

推荐使用以下结构编写提示词：

```markdown
# 角色定义
你是一个专业的客服助手...

## 能力范围
- 解答产品相关问题
- 处理售后投诉...

## 行为准则
- 保持礼貌和专业
- 不确定时诚实回答...

## 输出格式
请按以下格式回复：
1. 问题理解
2. 解决方案
3. 后续建议
```

### 关键要素

- **角色定位**：明确智能体的身份和专业领域
- **能力边界**：清晰界定能做什么、不能做什么
- **行为规范**：设定交互风格和约束条件
- **输出要求**：规定回复格式和质量标准

### 维护建议

- 定期根据实际对话效果优化模板
- 使用分类区分不同场景的模板
- 在描述中注明模板适用场景和注意事项
