# 对话界面

对话界面是与智能体交互的核心入口，提供流式对话、文件上传、工具调用确认、记忆控制等能力，帮助用户高效地与智能体进行多轮交互。

## 功能概述

- 支持流式对话，实时查看智能体的回复过程
- 支持多会话管理，包括新建、切换、重命名、置顶、删除
- 支持文件上传和预览，文件类型受智能体配置限制
- 支持记忆持久化开关，保持多轮对话上下文连贯
- 支持工具调用过程展示，以及需要确认的工具调用审批
- 支持工作空间面板（需智能体配置代码执行环境）
- 响应式布局，适配桌面端和移动端

## 界面布局

对话界面采用三栏布局结构：

| 区域 | 说明 | 是否必须 |
|------|------|----------|
| **左侧边栏** | 会话列表管理区域，包含置顶会话和历史记录 | 是 |
| **中间主区域** | 消息展示与输入区域，包含欢迎页、消息列表、输入框 | 是 |
| **右侧工作空间** | 代码执行工作空间面板（需智能体配置代码执行环境） | 否 |

![对话界面整体布局占位](images/chat-layout.png)

## 操作指南

### 开启新对话

1. 点击左侧边栏顶部的 **开启新对话** 按钮
2. 进入欢迎页，输入框中输入消息
3. 发送第一条消息后，系统自动创建新会话，并以消息内容生成会话标题

:::tip
发送第一条消息时，系统自动以消息前 50 个字符作为会话标题。你也可以在后续操作中重命名会话。
:::

![欢迎页占位](images/chat-welcome.png)

### 发送消息

在输入框中输入消息后，可通过以下方式发送：

| 操作方式 | 说明 |
|----------|------|
| **Enter** | 直接发送消息（仅桌面端） |
| **Shift + Enter** | 输入换行（仅桌面端） |
| **发送按钮** | 点击输入框右下角箭头按钮发送（桌面端和移动端通用） |

:::tip
移动端不响应 Enter 键发送，请使用发送按钮，避免虚拟键盘误触。
:::

### 中止生成

在智能体回复过程中，发送按钮会变为中止按钮（方形图标），点击即可中止当前生成。

中止后，已生成的部分内容会自动保存为一条助手消息。

### 上传文件

1. 点击输入框工具栏中的 **附件** 按钮（回形针图标）
2. 选择一个或多个文件
3. 文件会先显示为"上传中"状态，上传完成后显示文件名
4. 可点击文件项进行预览（支持图片等媒体类型）
5. 点击文件项右侧的 **x** 按钮可移除已上传的文件

![文件上传占位](images/chat-file-upload.png)

:::warning
文件类型受智能体配置限制，仅允许上传智能体配置中允许的文件类型。如果上传了不允许的文件类型，系统会提示警告并跳过该文件。
:::

### 工具栏功能

输入框下方工具栏提供以下功能按钮：

| 按钮 | 说明 | 启用条件 |
|------|------|----------|
| **记忆** (时钟图标) | 开启/关闭记忆持久化，开启后智能体可在多轮对话中保持上下文连贯 | 智能体需启用记忆功能 |
| **工具调用历史** (闪电图标) | 开启/关闭工具调用过程展示，开启后对话中会显示工具调用详情 | 智能体需开启"显示工具调用过程" |
| **附件** (回形针图标) | 上传文件附件 | 智能体需配置允许的文件类型 |
| **发送/中止** (箭头/方块图标) | 发送消息或中止生成 | - |

![输入框工具栏占位](images/chat-toolbar.png)

:::tip
- 记忆和工具调用历史的开关状态会自动持久化，下次进入同一智能体的对话界面时保持上次的选择
- 未启用的按钮显示为灰色不可点击状态，表示当前智能体不支持该功能
:::

## 会话管理

### 会话列表

左侧边栏展示当前智能体的所有会话，分为两个区域：

| 区域 | 说明 |
|------|------|
| **置顶** | 已置顶的会话，最多占侧边栏 50% 高度，超出可滚动 |
| **历史记录** | 未置顶的会话，支持滚动加载更多 |

![会话列表占位](images/chat-session-list.png)

### 侧边栏折叠

- 点击侧边栏顶部的 **折叠** 按钮，可收起侧边栏
- 折叠后仅显示图标按钮，点击 **展开** 按钮可恢复
- 侧边栏的折叠状态会自动持久化

### 切换会话

点击会话列表中的会话项即可切换到该会话，加载其历史消息。

:::warning
智能体正在回复时无法切换会话，需先等待回复完成或中止生成。
:::

### 会话操作

点击会话项右侧的 **更多** 按钮（三个点），可执行以下操作：

| 操作 | 说明 |
|------|------|
| **重命名** | 修改会话标题，在弹窗中输入新标题后点击确定 |
| **置顶** | 将会话固定到置顶区域，方便快速访问 |
| **取消置顶** | 取消固定，会话回到历史记录区域 |
| **删除** | 删除会话及其中所有消息，删除后无法恢复 |

:::warning
删除会话是不可逆操作，删除后所有消息将无法恢复，请谨慎操作。
:::

### 加载更多

历史记录区域支持滚动加载：滚动到列表底部时自动加载下一页会话，加载完成时显示"已经到底了"提示。

## 消息展示

### 用户消息

用户发送的消息显示在对话区域的右侧，包含：

- **文件附件**：如有上传文件，在消息气泡中展示文件列表，点击可预览
- **文本内容**：消息的文字内容

### 助手消息

智能体的回复显示在对话区域的左侧，支持 Markdown 渲染，包括：

- 代码块语法高亮
- 表格、列表等富文本格式
- 数学公式（如果智能体返回了公式内容）

### 工具调用

当智能体执行工具调用时，对话区域会显示工具调用进度：

| 状态 | 说明                             |
|------|--------------------------------|
| **调用中** | 显示"正在执行 [工具名]"，带加载动画           |
| **需要确认** | 显示工具名、参数（可展开/折叠），以及"允许"和"禁止"按钮 |
| **已完成** | 调用结果已返回并展示                     |

#### 工具调用确认

当工具标记为"需要确认"时，用户需要审批工具执行：

- 点击 **允许**：执行工具调用，将结果返回给智能体继续处理
- 点击 **禁止**：拒绝工具调用，通知智能体用户拒绝了本次请求
- 点击 **展开参数**：查看工具调用的参数详情

:::tip
工具调用确认功能仅在智能体配置中开启了"显示工具调用过程"时生效。
:::

### 错误消息

当智能体运行出错时，错误信息会以红色文字显示在对话区域中。

## 工作空间面板

当智能体配置了代码执行环境时，对话界面右上角会出现 **工作空间** 按钮（文件夹图标）：

- 点击按钮可展开/收起工作空间面板
- 面板从右侧滑出，与对话区域并排显示
- 进入欢迎页（新建对话）时，工作空间面板会自动关闭

![工作空间面板占位](images/chat-workspace.png)

## 智能体能力开关

对话界面中的部分功能受智能体配置影响，只有智能体启用了对应能力，对应功能才可用：

| 功能 | 智能体配置项 | 说明 |
|------|-------------|------|
| 记忆持久化 | 启用记忆 | 开启后可在工具栏切换记忆开关 |
| 工具调用历史 | 显示工具调用过程 | 开启后可在工具栏切换工具调用显示 |
| 文件上传 | 允许上传文件类型 | 配置后附件按钮可用 |
| 工作空间 | 执行环境配置 | 配置后工作空间面板入口可用 |

## 常见问题

### 发送消息没有反应？

- 检查输入框是否为空（纯空格不算有效内容）
- 检查是否有文件正在上传中（上传中的文件会阻止发送）
- 检查智能体是否正在回复中（需等待或中止后才能发送新消息）

### 文件上传失败？

- 检查文件类型是否在智能体允许的范围内
- 检查网络连接是否正常
- 文件上传失败后，该文件会自动从附件列表中移除

### 记忆开关是灰色的？

- 表示当前智能体未启用记忆功能
- 需在智能体配置中开启"启用记忆"选项

### 工作空间按钮不可见？

- 需在智能体的高级设置中配置"执行环境配置"
- 只有配置了代码执行环境的智能体才会显示工作空间入口
- 在欢迎页（无会话）时，工作空间入口也会隐藏

### 工具调用一直显示"正在调用"？

- 可能是工具执行耗时较长，请耐心等待
- 如长时间无响应，可点击中止按钮终止当前生成

### 移动端侧边栏如何关闭？

- 点击侧边栏外的遮罩层区域即可关闭
- 选择会话后侧边栏也会自动关闭
