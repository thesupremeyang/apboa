# 常见问题 (Q&A)

本章节汇总了平台使用过程中的常见问题及解决方案，帮助您快速定位和解决问题。

## 一、智能体相关

### 1、智能体配置的 Skill 中包含脚本，为什么无法执行？

:::info 原因
智能体执行脚本需要执行环境配置。
:::


:::success 解决方案
进入 **智能体 → 设计 → 配置 → 高级设置**，配置「执行环境配置」
:::


### 2、给智能体配置了敏感词，为什么不生效？

:::info 原因
配置敏感词后还需要手动启用敏感词过滤功能。
:::

:::success 解决方案
进入 **智能体 → 设计 → 配置 → 工具与能力**，勾选「启用敏感词过滤」选项
:::

### 3、如何配置智能体执行工具时需要用户确认？

:::success 解决方案
1. **开启工具确认**：进入 **工具 → 编辑**，开启「是否需要确认」选项
2. **配置确认钩子**：进入 **智能体 → 设计 → 配置 → 工具与能力 → 钩子配置**，选择内置钩子中的 `IConfirmationHook`
3. **启用记忆持久化**：进入 **智能体 → 设计 → 配置 → 高级设置**，勾选「启用记忆持久化」选项
4. **刷新对话页面**
:::

:::warning 提醒
以上四个步骤缺一不可，否则工具确认功能无法正常工作。
:::


### 4、如何关闭对话过程中的工具调用历史？

:::success 解决方案
1. 进入 **智能体 → 设计 → 配置 → 高级设置**，关闭「显示工具调用历史」选项
2. 刷新对话页面
:::


### 5、为什么多个 Agent 绑定同一 Studio 时，A 项目的消息出现在 B 项目中？

:::info 原因
AgentScope 中的 `StudioManager` 采用全局单例设计，其定位是调试智能体使用，不适合多个智能体同时开启。
:::

:::success 解决方案
智能体正式使用时，务必关闭 Studio 可视化调试功能，避免不同智能体之间的消息串扰
:::

### 6、如何停止智能体的定时任务？

:::success 解决方案
进入 **智能体 → 设计 → 定时**，取消「状态」开关即可暂停定时任务（任务配置保留，可随时重新启用）
:::

### 7、如何删除智能体的定时任务？

:::success 解决方案
进入 **智能体 → 设计 → 定时**，点击左下角的「解绑定时任务」按钮，即可彻底删除定时任务
:::

### 8、工具开启了“否需要确认”选项，为什么智能体调用工具时还是无需用户确认就自动执行了？

:::info 原因
Agent 调用工具实现用户确认，需要配合 IConfirmationHook 钩子使用
:::

:::success 解决方案
进入 **智能体 → 设计 → 配置 -> 工具与能力**，选择内置钩子中的 `IConfirmationHook`，即可实现工具调用时的用户确认
:::


### 9、多节点部署，上传技能包，为什么只能上传到某个节点目录下？

:::info 原因
平台并没有做多节点指定文件夹的自动更新实施同步
:::

:::success 解决方案
可以使用 GlusterFS 实现多节点服务器上指定文件夹的自动更新实施同步
:::

## 二、MCP 相关

### 1、配置的 MCP 不可使用？

:::success 解决方案
1. 检查 MCP Key 是否正确。进入 MCP 配置页面，确认 API Key 或认证信息无误
2. 如果配置的是 STDIO 协议，检查本地运行环境是否正确：
   - 确认本地已安装 MCP 服务所需的运行时环境（如 Node.js、Python 等）
   - 确认 MCP 服务的启动命令和参数是否正确
   - 确认本地环境变量配置是否完整
:::

## 三、对话相关

### 1、与智能体对话时，为什么无法上传文件？

:::success 解决方案
1. **配置存储方案**：进入 **系统设置 → 存储配置**，配置存储方案（推荐使用 `LOCAL` 本地存储，存储路径如 `E:\tmp\apboa`）
2. **配置多模态模型**：选择一个支持多模态的模型（如 `qwen3.5-plus`），在模型配置中勾选对应的「模型类型」（支持多选，如 IMAGE）
3. **为智能体配置多模态模型**：编辑智能体，将模型配置替换为支持多模态的模型，保存后即可在对话中上传文件
:::

### 2、如何在外部浏览器中打开对话页面（无需登录）？

:::success 解决方案
进入 **智能体 → 设计 → API**，复制「外置对话入口」中的地址，即可在外部浏览器中直接打开对话页面，无需登录即可使用
:::

### 3、如何对接平台中对话界面相关的 API？

:::success 解决方案
进入 **智能体 → 设计 → API**，重点关注以下内容：

- **智能体对话接口**：发送消息、接收回复的核心接口
- **鉴权方式**：接口认证机制（如 SkKey、ChatKey 等）
- **会话管理接口**：创建、查询、删除会话等管理操作

详细的接口文档和调用示例可在 API 页面中查看。
:::
