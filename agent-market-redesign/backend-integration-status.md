## 一、已对接后端 ✅

### 1. 智能体列表

| 项目 | 说明 |
|------|------|
| **接口** | `GET /api/agent/definition/page` |
| **调用方式** | `agentApi.page(params)` |
| **功能** | 分页加载智能体列表，支持 `page`、`size`、`enabled` 参数 |
| **前端位置** | 主内容区 - 智能体卡片网格 |

### 2. 搜索智能体

| 项目 | 说明 |
|------|------|
| **接口** | `GET /api/agent/definition/page?name=关键词` |
| **调用方式** | `agentApi.page({ name: query })` |
| **功能** | 输入关键词后防抖 300ms，通过 `name` 参数做服务端模糊搜索（LIKE 匹配） |
| **前端位置** | 主内容区顶部搜索栏 |

### 3. 标签筛选

| 项目 | 说明 |
|------|------|
| **接口** | `GET /api/agent/definition/get/tags` |
| **调用方式** | `agentApi.listTags()` |
| **功能** | 获取所有智能体的标签列表，左栏动态展示，点击可筛选 |
| **前端位置** | 左侧栏 - 标签筛选区域 |

### 4. 按标签过滤智能体

| 项目 | 说明 |
|------|------|
| **接口** | `GET /api/agent/definition/page?tag=标签名` |
| **调用方式** | `agentApi.page({ tag: tagName })` |
| **功能** | 点击标签后通过 `tag` 参数做服务端精确筛选，同时客户端二次过滤 |
| **前端位置** | 左侧栏标签按钮 → 主内容区卡片列表联动 |

### 5. 分页加载更多

| 项目 | 说明 |
|------|------|
| **接口** | `GET /api/agent/definition/page?page=N&size=20` |
| **调用方式** | `agentApi.page({ page: currentPage + 1 })` |
| **功能** | 点击"加载更多"按钮追加下一页数据，基于后端分页（`current < pages` 判断是否还有更多） |
| **前端位置** | 卡片网格底部 |

### 6. 智能体体验（跳转对话）

| 项目 | 说明 |
|------|------|
| **接口** | 无直接 API 调用，跳转到 `#/chat/{agentId}` |
| **功能** | 点击卡片或"立即体验"按钮，打开新窗口进入该智能体的对话页面 |
| **前端位置** | 卡片底部"立即体验"按钮 / 详情弹窗"立即体验"按钮 |

## 三、对接优先级建议

### P0 - 高优先级（影响核心体验）

1. **智能体卡片附加字段** - 评分、使用量、满意度、发布者，直接影响卡片信息展示
2. **仪表盘统计** - 页面首屏核心数据，当前为假数据影响可信度
3. **最新发布列表** - 右侧栏核心内容，可直接复用已有接口

### P1 - 中优先级（提升功能完整性）

4. **分类列表及计数** - 左侧栏导航核心，需后端新增分类表或从 tag 聚合
5. **排序功能增强** - 将客户端排序改为服务端排序
6. **筛选条件** - 场景、能力标签等多维筛选

### P2 - 低优先级（锦上添花）

7. **市场洞察 - 热门能力排名** - 需要统计分析能力
8. **使用趋势图** - 需要时序数据采集
9. **导航项页面** - 我的智能体、应用管理等独立页面
10. **发布智能体流程** - 完整的智能体创建向导
