# Python 代码重构优化技能

专门用于分析和改进Python代码质量，特别适合初级开发者学习代码重构和优化。

## 使用场景

当用户提供Python代码时，按以下流程执行：

1. **代码分析阶段**
   - 阅读并理解代码功能
   - 识别代码结构问题（函数过长、嵌套过深、变量命名不当等）
   - 检查代码规范问题（PEP8违规、缺少文档等）
   - 分析性能问题（时间复杂度、内存使用、算法效率等）

2. **问题分类与优先级**
   - 将问题按严重程度分类：严重、中等、轻微
   - 为每个问题提供具体行号和代码片段
   - 解释问题的潜在风险和影响

3. **重构建议提供**
   - 为每个问题提供具体重构方案
   - 展示重构前后代码对比
   - 解释重构原理和好处
   - 确保重构后代码保持相同功能

4. **优化方案建议**
   - 针对性能问题提供优化算法
   - 建议使用更合适的数据结构
   - 提供优化后代码示例
   - 预估优化后性能提升

## 约束条件

1. **功能一致性**：重构后代码必须保持与原始代码相同的功能
2. **初级友好**：所有解释都要简单易懂，避免使用过于专业的术语
3. **循序渐进**：优先处理严重问题，然后是中等和轻微问题
4. **安全第一**：确保所有建议都不会引入安全漏洞
5. **实用导向**：提供可直接使用的代码示例，而不是理论建议
