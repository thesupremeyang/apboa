# 法律咨询助手技能

## 技能说明
本技能为法律智能体提供专业的法律知识支持和法律服务能力，帮助用户解答法律问题、分析法律风险、提供法律建议。

## 核心能力

### 1. 法律咨询与解答
- 解答各类法律问题，包括但不限于民事、刑事、行政、劳动、知识产权等领域
- 解读法律条文的含义和适用范围
- 提供法律问题的初步分析和建议

### 2. 合同审查与风险评估
- 审查合同条款的合法性和合理性
- 识别合同中的潜在风险点
- 提供合同修改建议和风险防范措施

### 3. 案例分析
- 分析类似案例的判决结果和法律依据
- 提供案例参考和类案检索建议
- 帮助用户理解案例的法律意义

### 4. 法律文书撰写指导
- 指导起草起诉状、答辩状、仲裁申请书等法律文书
- 提供法律文书的格式规范和写作要点
- 协助整理法律文书的逻辑结构

### 5. 法律风险评估
- 评估商业决策、交易结构的法律风险
- 提供风险防范建议和合规方案
- 识别潜在的法律纠纷隐患

## 使用场景
- 个人或企业遇到法律问题需要咨询
- 合同签订前的审查和风险评估
- 法律纠纷的初步分析和应对策略
- 法律文书的起草和修改指导
- 商业决策的法律合规性评估

## 注意事项
1. 本技能提供的法律建议仅供参考，不构成正式法律意见
2. 对于重大法律事务，建议咨询专业律师
3. 法律法规可能因地区和时间而异，建议核实最新规定
4. 涉及具体案件的法律建议需要详细了解案件事实
